Instructions ENGLISH

This extension is very customizable. It can be used to show nice photos linkable or banner ads. Nivo Slider on the home of your forum.
- Many images loadable
- Effects of random transition
- Images linkable
- Label images
- Responsive
- Change image evry 5 seconds
- Direction Navigation

To use this extension by default, simply overwrite the images. Will be automatically adjusted, If you want to add links to the images the file to edit is: overall_header_page_body_before.html
Adding a title: "Title =" Image Name "will add the comment rimmed black to the slider.

Instructions Italiano

Questa estensione è molto personalizzabile. Si può utilizzare per mostrare delle belle foto linkabili o per dei banner pubblicitari. Nivo Slider per il vostro forum.
- Immagini a scelta caricabili nello slidershow
- Effetti casuali di transizione
- Immagini con link esterni
- Responsive per dispositivi mobile
- Cambia le immagini ogni 5 secondi
- Pannello di direzione e navigazione

Per usare questa estensione di default, basta semplicemente sovrascrivere le immagini con quelle che volete. Se volete aggiungere link alle vostre immagini, basta editare il file: overall_header_page_body_before.html
Aggiungendo un nome al tag: title="Nome Immagine" aggiungerete una scritta al commento delle immagini nello slidershow (Stringa nera).

Per qualsiasi altra informazione potete contattarci sul nostro forum in Italiano.
